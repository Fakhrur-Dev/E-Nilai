# E-Nilai

> **Cloud Academic Assessment System**
>
> **Sederhana untuk Guru, Cerdas untuk Data**

---

## Tentang E-Nilai

E-Nilai adalah aplikasi web modern untuk membantu guru mengelola nilai peserta didik secara sederhana, cepat, aman, dan responsif.

Berbeda dengan aplikasi penilaian pada umumnya, **E-Nilai tidak menyimpan data akademik siswa di server developer**.

Seluruh data nilai, master data, dan arsip akademik disimpan langsung pada **Google Drive milik pengguna**, sehingga kepemilikan data sepenuhnya berada di tangan guru atau sekolah.

---

## Filosofi

E-Nilai dikembangkan berdasarkan beberapa prinsip utama.

- ✅ Data tetap milik pengguna
- ✅ Antarmuka sederhana seperti spreadsheet
- ✅ Responsif di Desktop, Tablet, Android, dan iPhone
- ✅ Batch Save (bukan realtime setiap perubahan sel)
- ✅ Modular dan mudah dikembangkan
- ✅ Aman menggunakan Google Authentication

---

## Fitur Utama

### Authentication

- Google Sign-In
- Firebase Authentication
- Aktivasi Lisensi Free & Premium

### Workspace

- Workspace otomatis
- Folder Google Drive otomatis
- Google Sheets otomatis
- Backup & Restore

### Master Data

- Sekolah
- Guru
- Tahun Ajaran
- Semester
- Mata Pelajaran
- Kelas
- Data Siswa (NIS & NISN)

### Penilaian

- Input nilai seperti spreadsheet
- Target Pembelajaran (TP)
- KKM
- Nilai Rapor
- Riwayat perubahan nilai

### Laporan

- Rekap nilai
- Deskripsi kompetensi
- Analisis capaian TP
- Ranking dinamis (tidak disimpan)
